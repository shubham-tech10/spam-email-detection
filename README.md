# Spam Email Detection

## Project Overview
This project uses Natural Language Processing (NLP) and Machine Learning to classify SMS messages as Spam or Ham.

## Technologies Used
- Python
- Pandas
- Scikit-Learn

## Algorithm
- CountVectorizer
- Multinomial Naive Bayes

## Workflow
1. Load Dataset
2. Data Exploration
3. Label Encoding
4. Text Vectorization
5. Train-Test Split
6. Model Training
7. Prediction
8. Evaluation

## Result
Accuracy: 98.4%
